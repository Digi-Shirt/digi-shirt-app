# Digi-Shirt

## Quick Start

1.  See prerequisite tools 
2.  Clone this repo (and cd into the digi-shirt-app directory)
3.  `yarn install`
4.  `npm start` or `expo start`

## Prerequisite tools

- npm & node
- yarn
- Android Studio
- Simulator (for iOS emulation; only available on OSX)
